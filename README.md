# 基于Pug+ Koa2+Mongodb实现多人共享博客v1

技术栈(v1)：仿照**layui**社区界面，利用**pug**模版在服务端渲染，使用Nodejs（koa）作为后台，MongoDB作为数据库

- [x] 登录注册 
- [x] 发表文章
- [x] 发表评论
- [x] 上传头像
- [x] 后台删除文章，评论
- [x] （管理员）后台删除用户

技术栈 (v2): 使用antd重构页面，前后端分离  线上预览地址 http://mrshulan.xin/blog/

> - [x] 页面重构
> - [x] 新增点赞功能
> - [x] 支持markdown编辑
